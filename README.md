# 金融财经自动化报告系统

自动化金融财经邮件报告系统，基于 GitHub Actions 云端定时执行。

## 功能

| 报告 | 时间 | 说明 |
|------|------|------|
| 综合市场报告 | 每日 08:00 | 全球市场概览 + AI分析 |
| 美股简报 | 每日 22:25 | 美股三大指数 + 科技股 |
| 油价报告 | 每12小时 | WTI/布伦特原油 |
| 海湾战争简报 | 每4小时 | 地区局势速报 |
| 美元指数报告 | 每12小时 | 美元指数 + 黄金 |
| A股分析 | 周六 07:30 | A股市场分析 |
| 每周市场报告 | 周六 10:30 | 一周市场总结 |
| 国际媒体海湾 | 每日 09:33 | 国际媒体报道 |

## 配置

在 GitHub 仓库设置中添加以下 Secrets：

- `DEEPSEEK_API_KEY` - DeepSeek API密钥
- `SENDER_EMAIL` - 发件人邮箱
- `SENDER_PASSWORD` - 邮箱授权码
- `RECEIVER_EMAIL` - 收件人邮箱
- `RECEIVER_EMAIL_2` - 第二收件人邮箱（可选）

## 手动触发

在 GitHub Actions 页面点击 "workflow_dispatch" 可手动执行任务。
